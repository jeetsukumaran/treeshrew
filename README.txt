treeshrew is a C++ (2011) program for the Bayesian estimation of alignments,
trees, and evolutionary parameters from short-read sequence data.

Build dependencies:

    $ ./build-deps

Before first building:

    $ ./bootstrap.sh

Building (using local dependencies)

    $ mkdir -p build/debug
    $ cd build/debug
    $ ../../cfgcommand.sh
    $ make
